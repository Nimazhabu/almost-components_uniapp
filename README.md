## almost-components for uni-app
该仓库是一个用于 `uni-app` 框架的组件集合

`uni-app` 是一个使用 [Vue.js](https://github.com/vuejs/vue) 开发所有前端应用的框架，开发者编写一套代码，可发布到iOS、Android、H5、以及各种小程序（微信/支付宝/百度/头条/QQ/钉钉/淘宝）、快应用等多个平台。
